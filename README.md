# Habbitapp

## Run Instructions

- Fork and clone repo
- `cd server`
  - `npm install`
  - `npm start`
- `cd client`
  - `npm install`
  - `npm start`
- Make sure to have a PSQL database called `habitapp`
- `cd server`
- `\i db database.sql`
- `npm start`
