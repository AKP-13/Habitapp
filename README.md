# Habbitapp

## Run Instructions

- Fork and clone repo
- `cd` into `server`
  - `npm install`
  - `npm start`
- `cd` into `client`
  - `npm install`
  - `npm start`
- Make sure to have a PSQL database called `habitapp`
- `cd` into `server`
- `\i db database.sql`
- In new repo run `npm start`
